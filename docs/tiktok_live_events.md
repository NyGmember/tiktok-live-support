# TikTokLive Event Attributes

This document lists the attributes and properties of various events in the TikTokLive library, generated by inspection.

## CommentEvent

| Attribute | Type                                                   | Description               |
| :-------- | :----------------------------------------------------- | :------------------------ |
| `user`    | `<class 'TikTokLive.proto.custom_proto.ExtendedUser'>` | User who sent the comment |
| `comment` | `str`                                                  | The comment text          |

## GiftEvent

| Attribute      | Type                                                   | Description                                  |
| :------------- | :----------------------------------------------------- | :------------------------------------------- |
| `user`         | `<class 'TikTokLive.proto.custom_proto.ExtendedUser'>` | User who sent the gift                       |
| `gift`         | `<class 'TikTokLive.proto.custom_proto.ExtendedGift'>` | The gift details                             |
| `repeat_count` | `int`                                                  | Number of times the gift was sent in a combo |
| `repeat_end`   | `bool`                                                 | Whether the combo has ended                  |

## LikeEvent

| Attribute     | Type                                                   | Description               |
| :------------ | :----------------------------------------------------- | :------------------------ |
| `user`        | `<class 'TikTokLive.proto.custom_proto.ExtendedUser'>` | User who sent the likes   |
| `count`       | `int`                                                  | Number of likes sent      |
| `total_likes` | `int`                                                  | Total likes in the stream |

## ConnectEvent

| Attribute   | Type  | Description                  |
| :---------- | :---- | :--------------------------- |
| `unique_id` | `str` | The uniqueId of the streamer |
| `room_id`   | `int` | The room ID                  |

## DisconnectEvent

| Attribute        | Type | Description |
| :--------------- | :--- | :---------- |
| (No annotations) | -    | -           |

## ShareEvent

| Attribute | Type                                                   | Description                |
| :-------- | :----------------------------------------------------- | :------------------------- |
| `user`    | `<class 'TikTokLive.proto.custom_proto.ExtendedUser'>` | User who shared the stream |

## FollowEvent

| Attribute | Type                                                   | Description       |
| :-------- | :----------------------------------------------------- | :---------------- |
| `user`    | `<class 'TikTokLive.proto.custom_proto.ExtendedUser'>` | User who followed |

## JoinEvent

| Attribute | Type                                                   | Description     |
| :-------- | :----------------------------------------------------- | :-------------- |
| `user`    | `<class 'TikTokLive.proto.custom_proto.ExtendedUser'>` | User who joined |

## ViewerUpdateEvent

| Attribute      | Type  | Description               |
| :------------- | :---- | :------------------------ |
| `viewer_count` | `int` | Current number of viewers |

_(Note: Full attribute list for all events is available in the inspection output log)_
